- 两个工具: SEARCH_NOTE("keyword") 搜索全文, DEFINE("term") 查医学词典
- 两轮提取设计: Pass 1 正常提取 → 检测 tool calls → Pass 2 注入工具结果重提取
- 零额外开销: 不需要工具时直接用 Pass 1 结果
- Gate 3/4 跳过 tool-enriched 字段 (防止基于 A/P context 的验证覆盖全文信息)

### 验证结果
- ROW 12 Advance care: "Not discussed" → "Patient has completed POLST and is DNR/DNI" ✅
- 模型自主决策何时使用 SEARCH_NOTE (Advance care 搜索 DNR/DNI/Code status 等)
- Gate 3 IMPROVE 曾覆盖 tool 结果 → 修复为跳过 tool-enriched 字段
- Gate 4 FAITHFUL 也跳过 tool-enriched 字段 (信息来源是全文不是 A/P)
